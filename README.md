# 🎮 GameHub

Коллекция классических веб-игр с современным дизайном.

## 🕹️ Игры

| Игра | Игроки | Описание |
|------|--------|----------|
| 🐍 Змейка | 1 | Классическая аркада с сохранением рекорда |
| ❌⭕ Крестики-нолики | 1-2 | PvP или против ИИ (minimax) |
| ⬤ Шашки | 2 | Русские шашки на двоих |
| 🏓 Понг | 2 | Аркада на одной клавиатуре |

## 🚀 Запуск

Открой `index.html` в браузере или [играй онлайн](https://YOUR_USERNAME.github.io/gamehub/)

## 🛠 Технологии

- Чистый HTML/CSS/JS — никаких зависимостей
- Canvas API для рендеринга
- Minimax алгоритм для ИИ
- LocalStorage для сохранения рекордов
- CSS анимации и glassmorphism дизайн
